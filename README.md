Embed chat in your website in less than 5 minutes.

## Install

1. [Redis](https://redis.io/download#installation)
2. [Node.js](https://nodejs.org/en/)
3. `git clone https://github.com/super3/embed.chat`
4. `npm install`
5. `node server`
